# Strumenti per Insegnanti

Gli insegnanti gestiscono le loro classi dalla [dashboard insegnante](/dashboard). Gli strumenti sono progettati per essere rapidi e diretti.

## Assegnare le Lezioni

1. Apri la [pagina di assegnazione](/dashboard/assign/[lessonId]).
2. **Seleziona Studenti:** Usa la ricerca e i filtri per trovare gli studenti giusti. Puoi effettuare selezioni multiple.
3. **Imposta Date:** Scegli una Data di Inizio e una Scadenza.
4. **Notifiche:** Decidi quando gli studenti devono essere informati.
5. **Invia:** Clicca per assegnare.

## Valutare le Consegne

1. Apri la pagina di [correzione](/dashboard/grade/[assignmentId]).
2. **Revisione:** Controlla le risposte degli studenti.
3. **Feedback:** Aggiungi commenti a domande specifiche o un feedback generale per l'intero compito.
4. **Punteggio:** Assegna un voto. Questo aggiorna istantaneamente la dashboard e i punti dello studente.

## Gestire le Lezioni

- **Panoramica:** L'elenco delle lezioni ti mostra quanti studenti sono stati assegnati a ciascuna lezione.
- **Difficoltà:** Imposta i livelli di difficoltà usando il misuratore visivo a barre.
- **Condivisione:** Genera link pubblici di condivisione per le tue lezioni.

## Consigli per la Creazione di Contenuti

- **Libretti di Istruzioni:** Non riscrivere sempre le stesse istruzioni. Crea un "Libretto" e inseriscilo in qualsiasi compito con un clic.
- **Audio:** Usa il caricatore del feed di SoundCloud per recuperare rapidamente le tue ultime tracce.
- **Importazioni CSV:** La maggior parte dei creatori (Flashcard, Scelta multipla) supporta il caricamento via CSV. Cerca il pulsante con l'icona per importare dati in blocco.
- **Testi Sincronizzati:** Carica un file audio e un file `.lrc`. LessonHub li sincronizzerà automaticamente per gli studenti.

## Gestire le Hub Guides

- **Bozze:** Imposta `guideIsVisible` su `false` mentre stai lavorando a una guida.
- **Contenuti Gratuiti:** Imposta `guideIsFreeForAll` su `true` per offrirli a tutti gratuitamente.
- **Aggiornamenti:** Ogni modifica apportata viene registrata con una marca temporale, così gli studenti sanno che ci sono nuovi contenuti.
