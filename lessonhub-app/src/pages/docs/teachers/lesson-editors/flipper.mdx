# Flipper Lesson Editor

Use the Flipper lesson type to create a memory-style matching game where students pair an image tile with its word.

## Required

- **Lesson title** and **lesson preview**.
- **12 tiles**: upload 12 same-size images and add the matching word for each tile.
- **Attempts before penalty**: set how many failed attempts are allowed before penalties start (minimum 3).

## How it works for students

- Students flip two cards at a time to find matches.
- Attempts reset after each match.
- Rewards are based on attempts: attempt 1 = €10, attempt 2 = €5, attempt 3 = €1, then -€5 per extra attempt.
- Completed matches show a green check and lock.

## Tips

- Use clear, high-contrast images.
- Keep words short so they remain readable on mobile screens.
- Use similar image sizes to keep the grid clean.

## AI Prompt Template

Use this prompt to generate copy-paste ready content:

```text
Topic: <paste topic>
Level (CEFR A1-C2): <paste level>

Create a FLIPPER lesson for LessonHub using the topic and level above. Output ONLY the following 5 sections, exactly in this order, with no extra text:
Title:
Suggested cost:
Instructions:
CSV:
Additional information:

Rules:
- Suggested cost must be a teacher-charged price range in euros for a single 50-60 min online private language lesson in Italy. Use €15-€25 for standard/beginner lessons and €25-€40 for advanced/specialized/premium lessons. Choose one range and output it like "€18-€28".
- Use the topic and level to control vocabulary, grammar, and question difficulty.
- Instructions must be 1–3 short, practical sentences.
- CSV must be "N/A".
- Additional information must include 12 tiles as a list: each line should include the word and a short image description to guide image selection. Keep lines short.
```
